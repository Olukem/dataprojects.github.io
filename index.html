Steam Game Recommendation System using Spark and MLlib

This project leverages a dataset extracted from Steam, an online video game distribution service, to build a collaborative filtering recommendation system. The dataset, steam-200k.csv, contains user interactions with games, including purchases and playtime. Using Apache Spark and its MLlib library, load the dataset into a Spark DataFrame, conduct exploratory data analysis, preprocess the data, and train a recommendation model using Alternating Least Squares (ALS) matrix factorization. The project demonstrates the application of big data tools and machine learning techniques to create a practical recommendation system for a real-world dataset.

Hospital Database System Design and Implementation

As a database developer consultant, this project involves designing and implementing a new database system for a hospital to manage patient, doctor, medical record, appointment, and department information. The database will ensure data integrity, security, and efficient data retrieval. It includes Designing and normalizing the database into 3NF. Implementing constraints, stored procedures, user-defined functions, views, and triggers using T-SQL. Populating tables with sample data and performing various queries to meet the client's requirements. Providing recommendations on data integrity, concurrency, security, backup, and recovery. The project utilizes Microsoft SQL Server Management Studio for T-SQL implementations and includes comprehensive documentation of the design and implementation process.
